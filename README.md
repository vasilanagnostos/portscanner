## Port Scanner

* Create a database named <b>portscanner</b> and import the file db.sql.
* Modify the server list in <b>server_list</b> table

###To run:
* In Windows console:
  * <b>scan</b>

* In Linux console:
  * <b>./scan</b>
  * <b>./cronscan</b>
  
The results are stored in the table <b>server_results</b>
